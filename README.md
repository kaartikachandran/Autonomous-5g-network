# Autonomous 5G Network – Hardware RF Testbed for AI Resource Distribution

## Overview

Autonomous 5G Network is a hardware-based RF testbed designed to demonstrate intelligent resource distribution and network optimization using embedded systems, wireless communication, and AI-driven decision-making.

The project combines ECE concepts such as wireless communication, RF networking, latency analysis, and signal strength monitoring with software technologies including IoT, cloud databases, and machine learning.

## Objectives

* Simulate a small-scale autonomous wireless network
* Measure network performance metrics such as latency and RSSI
* Collect real-time communication data
* Enable intelligent resource allocation using AI algorithms
* Build a practical hardware testbed for future 5G research concepts

## Hardware Components

* ESP32 Base Station
* ESP32 Transmitter Nodes
* Wi-Fi Communication
* LCD Display Module
* Power Supply Unit

## Software Stack

* Arduino IDE
* ESP32 Firmware
* Firebase
* Python
* Machine Learning (Planned)
* NS3 Digital Twin (Planned)

## Current Implementation

### Base Station Node

* Receives packets from transmitter nodes
* Calculates latency
* Measures RSSI values
* Displays network statistics
* Uploads metrics to Firebase

### Transmitter Node

* Connects to Base Station
* Sends timestamped packets
* Receives acknowledgements
* Supports network performance monitoring
